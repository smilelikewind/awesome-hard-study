### 详细方案介绍

#### 1️⃣ Tushare Pro ⭐ A股基本面首选

**官网**: https://tushare.pro/  
**Python 包**: `pip install tushare`

**价格（积分制）**:

| 积分档位 | 约人民币 | 调用限制 | 说明 |
|---------|---------|---------|------|
| 免费版 | ¥0 | ~500次/分钟受限 | 基础数据，历史受限 |
| 2000积分 | ~¥200 | 200次/分钟，10万/天 | 基础日线 |
| **5000积分** | ~¥500 | 500次/分钟，无上限 | ✅ A股常规日线调用自由 |
| A股分钟数据 | ¥1000/月 | 500次/分钟 | 需单独订阅 |
| 港股实时数据 | ¥1000/月 | - | 仅日线 |

**数据覆盖**:
- ✅ A股 日线/行情/财务/资金流向/财报
- ✅ 指数、ETF、基金、期货
- ✅ 筹码分布、龙虎榜、涨跌停榜单
- ✅ 标准化 pandas DataFrame 格式

**⚠️ 风险提示**:
- 积分通胀（积分价值逐年下降）
- 2025年曾停运一周，稳定性风险
- 无 WebSocket，实时性弱（200-500ms）
- 分钟/港美股需单独付费

**示例代码**:
```python
import tushare as ts
pro = ts.pro_api('你的token')
df = pro.daily(ts_code='000001.SZ', start_date='20240101', end_date='20241231')
```

---

#### 2️⃣ AKShare ⭐ 免费神器（另类数据军火库）

**官网**: https://akshare.akfamily.xyz/  
**Python 包**: `pip install akshare`

**价格**: 🆓 **100% 免费开源**

**特点**:
- 本质是爬虫聚合，数据源包括：新浪财经、东方财富、同花顺等
- 数据覆盖极广，很多付费终端买不到的数据这里有
- **注意**：请求间隔建议 ≥3秒，避免被反爬

**数据覆盖**:
- ✅ 股票（A股/港股/美股）
- ✅ 期货（国内/国际）、期权、基金、债券、外汇、加密货币
- ✅ 宏观经济指标、行业数据、新闻舆情
- ✅ 提供 HTTP API 版本（AKTools），非 Python 用户也能用

**示例代码**:
```python
import akshare as ak
# 获取平安银行日线
stock_df = ak.stock_zh_a_hist(symbol="000001", period="daily")
# 获取龙虎榜
stock_board = ak.stock_lhb_gg_today_em(date="20241231")
```

## 选择决策指南

### 🎯 按用户类型推荐

#### 学生/入门学习者
1. **AKShare** - 完全免费，学习资源丰富，覆盖全面
2. **聚宽/米筐免费版** - 云端回测环境，无需搭建
3. **Tushare免费版** - 基础回测够用

#### 个人开发者/爱好者
1. **Tushare Pro (500积分)** - ¥500买断A股日线调用自由
2. **JQData试用后付费** - 数据质量稳定，本地调用方便
3. **TickDB / iTick API** - 如需实时WebSocket行情

#### 专业量化研究员
1. **Tushare Pro + Infoway API** - 回测+实时组合
2. **JQData / RQData正式版** - 机构级数据质量
3. **Choice / iFinD** - 性价比机构终端替代

#### 机构/高频交易
1. **Wind** - 行业标准，债券/宏观数据护城河
2. **恒生PTrade / 迅投QMT** - 券商托管低延迟柜台
3. **RiceQuant私有化部署** - 局域网低时延访问
4. **掘金量化** - C++/Python双语言，实盘毫秒级

#### 跨市场交易者（A股+美股+外汇+加密货币）
1. **TickDB** - 单API覆盖6类资产
2. **iTick API** - 三市场统一接入，WebSocket &lt;100ms延迟

### ⚠️ 选型核心原则（2026年）

1. **稳定性第一** - 永远准备 Plan B，双源切换是底线
2. **数据完整性** - 别信"全球覆盖"，自行测试冷门标的
3. **实时能力** - WebSocket优先，轮询仅适合低频
4. **性能** - 批量获取速度，并发能力
5. **价格投入产出比** - 先免费试用验证逻辑，再考虑付费
6. **文档与支持** - 示例代码能否跑通？错误处理是否标准化？

## 2026年市场趋势

新一代 API 服务商（TickDB, iTick, Infoway）正在通过跨市场统一接入、WebSocket支持、透明定价，抢占 Tushare/AkShare 的中高端用户。传统量化平台正加速向实盘交易通道方向转型。

跨境多资产需求增长，单一 A股 API 已无法满足现代策略需求。
